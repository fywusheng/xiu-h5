# 二维码生成器

### 介绍

js 生成二维码图片

## 代码演示

### 基础用法

```HTML
<y-QR-code :content="http://wxfwtest.ylzms.com" />
```

## Api

### Props

| 参数    | 说明       | 类型   | 可选值 | 默认值 |
| ------- | ---------- | ------ | ------ | ------ |
| content | 二维码内容 | string | -      | -      |
| width   | 宽度       | number | -      | 110    |
| height  | 高度       | number | -      | 110    |
